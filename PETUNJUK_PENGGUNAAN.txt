📌 LANGKAH INSTALL
============================================

1. INSTALL XAMPP
   - Download dari https://www.apachefriends.org/
   - Install seperti biasa (next next)
   - Buka XAMPP Control Panel
   - Start Apache dan MySQL

2. LETAKKAN FILE PROJECT
   - Copy folder "perpustakaan_dyzen"
   - Paste ke C:\xampp\htdocs\

3. IMPORT DATABASE
   - Buka http://localhost/phpmyadmin
   - Klik New > Buat database: perpustakaan_dyzen
   - Klik Import > Pilih file "database_lengkap.sql"
   - Klik Go

4. AKSES APLIKASI
   - Buka browser, ketik: http://localhost/perpustakaan_dyzen/
